[Ways We Travel](https://wayswe.travel) is a website I built for my spouse and I to document some of the places we've travelled, how we saw them, and what we did there at specific moments in time.

I built it in Astro, using Svelte for the more complex components and Three.js for the rotating globe logo. It's currently deployed with Cloudflare Pages.
